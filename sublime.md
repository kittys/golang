### install go
* https://golang.org/doc/install
* append path in .profile
```
export GOPATH=$HOME/go
export GOROOT=/usr/local/go/
export PATH=$PATH:$GOROOT/bin
```
* `. ~/.profile` to reload profile without logout
* create go file and try go build
